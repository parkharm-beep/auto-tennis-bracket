테니스 대진표 생성기 사용법

참가자 명단만 엑셀에 적으면, 클릭 한 번으로 코트·시간이 짜인 주간 대진표가 만들어집니다. 컴퓨터가 익숙하지 않아도 아래 순서대로만 하시면 됩니다.

⏱️ 딱 3단계 요약
  1. 빈 양식을 내려받아 참가자 명단을 채웁니다.
  2. 채운 파일을 올리고 대진표 생성 버튼을 누릅니다.
  3. 완성된 대진표 엑셀을 내려받습니다.
💡 PC(노트북·데스크탑) + 크롬(Chrome) 브라우저를 권장합니다. 휴대폰에서도 되지만 화면이 작고 조금 느립니다.

1사이트 접속하고 비밀번호 넣기

인터넷 주소창에 아래 주소를 칩니다(또는 받으신 링크를 누릅니다).

parkharm-beep.github.io/auto-tennis-bracket/

처음 보이는 비밀번호 입력 화면에 클럽에서 공유받은 비밀번호를 넣고 들어가기를 누릅니다.

※ 비밀번호는 클럽 회원만 들어오게 하는 잠금장치입니다. 모르면 총무님께 문의하세요.

2"준비 완료"가 뜰 때까지 기다리기

비밀번호를 통과하면 대진표 화면이 나옵니다. 화면 가운데 아래쪽 검은색 로그창에서 글자가 올라갑니다.

처음 한 번은 계산 도구를 내려받느라 5~10초 걸립니다(약 10MB). 로그창에 "준비 완료. 입력 엑셀을 선택하고…"라는 줄이 보이면 사용할 준비가 된 것입니다.

💡 두 번째부터는 저장(캐시)되어 거의 바로 시작합니다.

3입력 양식(빈 엑셀) 내려받기

화면 위쪽의 버튼을 누르면 빈 엑셀 양식이 다운로드됩니다.

빈 양식 다운로드아무것도 안 채워진 깨끗한 양식.
사전채움 양식 다운로드우리클럽 멤버 명단이 미리 채워진 양식(이름·성별·구력·IN/OUT 등). 보통 이걸 받아 그날 오는 사람만 남기고 시간만 손보면 됩니다.

다운로드된 파일은 보통 컴퓨터의 "다운로드" 폴더에 테니스_입력양식.xlsx 라는 이름으로 저장됩니다. 더블클릭해서 엑셀로 엽니다.

4양식 채우기 (가장 중요!)

엑셀 파일에는 보통 시트 두 개가 있습니다. 아래쪽 시트 탭을 눌러 둘 다 채웁니다.

① "참가자" 시트 — 사람 한 줄씩

칸무엇을 적나
번호1, 2, 3… 순번 (안 적어도 됨)
이름참가자 이름 (필수)
성별남 또는 여 (필수)
구력테니스 경력을 숫자로. 예: 3, 10 (필수)
구분정회원이면 정회원, 게스트면 게스트 선택
클럽평소엔 비워두세요. 다른 클럽과 교류전 할 때만 클럽명을 적습니다 (아래 설명)
IN시간코트에 들어올 수 있는 가장 빠른 시각. 예: 08:00 (30분 단위)
OUT시간코트를 떠나야 하는 시각. 예: 10:00 (반드시 IN보다 늦게)
최대게임수이 사람이 뛸 최대 게임 수. 비우면 제한 없음. 예: "3게임만" → 3
메모자유 메모 (안 적어도 됨)

② "코트" 시트 — 코트 한 줄씩

칸무엇을 적나
코트명코트 이름. 예: A, B, C
시작시간그 코트를 쓰기 시작하는 시각. 예: 07:00
종료시간그 코트 사용이 끝나는 시각. 예: 10:00
⚠️ 시간은 꼭 30분 단위로 적습니다(00분 또는 30분). 한 게임 = 30분입니다.
예) 08:00, 08:30, 09:00 … (08:15 같은 건 안 됩니다)

다 채웠으면 엑셀을 저장(덮어쓰기)합니다.

🤝 다른 클럽과 교류전을 할 때
참가자 시트의 "클럽" 칸에 각자 소속 클럽명을 적으세요. 예: 우리 회원은 우리클럽, 방문팀은 ○○클럽.
→ 그러면 같은 클럽끼리만 한 팀(복식 2명)이 되고, 경기는 우리 클럽 팀 vs 상대 클럽 팀으로 자동 구성됩니다.
• 클럽 칸을 모두 비우면 평소(클럽 구분 없음)대로 동작합니다.
• 어떤 시간대에 한 클럽 인원이 4명이 안 모이면 그 코트는 비게 될 수 있어요(규칙을 지키기 위해).

5채운 파일을 사이트에 올리기

다시 대진표 화면으로 돌아옵니다. "입력 엑셀" 칸의 파일 선택을 누르고, 방금 저장한 엑셀 파일을 고릅니다.

6옵션 확인 (그냥 둬도 됩니다)

아래 칸들은 비워두거나 기본값 그대로 둬도 잘 됩니다. 바꾸고 싶을 때만 건드리세요.

날짜대진표에 찍힐 날짜. 예: 26.5.30
시드대진 배치를 바꾸는 번호. 결과가 마음에 안 들면 숫자만 바꿔 다시 생성하면 다른 조합이 나옵니다.
반복 횟수더 좋은 조합을 찾는 시도 횟수. 기본 150이면 충분(약 3초). 키울수록 품질↑·시간↑.
제목대진표 맨 위에 들어갈 제목.

7대진표 만들기

파란색 대진표 생성 버튼을 누릅니다. 검은 로그창이 움직이며 몇 초간 계산합니다. 버튼이 "처리 중…"으로 바뀌었다가 끝나면 결과가 나옵니다.

8완성된 대진표 내려받기

맨 아래에 결과가 표시됩니다.

  • 초록색 테니스_대진표_…xlsx 다운로드 버튼 → 누르면 완성 파일이 저장됩니다.
  • 판정: PASS면 품질 양호, RETRY면 시드를 바꿔 다시 만들어 보길 권장.
  • 참가자·코트·매치 수, 게임수 균형, 경고/이슈 요약도 함께 보입니다.
📊 결과 엑셀에서 알아두면 좋은 점
• 오른쪽 사람별 게임수는 엑셀 수식(자동 계산)입니다. 대진표 칸의 이름을 직접 바꾸거나 지우면 게임수가 자동으로 다시 계산됩니다. (엑셀에서 열었을 때 작동)
• 교류전일 때는 오른쪽 명단이 클럽별로 좌·우 분리되어 표시되고, 클럽별 게임수 합계도 따로 나옵니다.

받은 엑셀을 열어 인쇄하거나 단체방에 공유하면 끝입니다. 🎉

❓ 자주 묻는 질문 / 문제 해결

화면이 안 뜨고 "404"가 나와요

잠깐 점검 중일 수 있습니다. 주소가 정확한지 확인하고, Ctrl + Shift + R(강력 새로고침)을 눌러 보세요. 그래도 안 되면 관리자에게 알려주세요.

비밀번호가 틀렸다고 나와요

대소문자·공백까지 정확히 입력해야 합니다. 클럽에서 공유받은 비밀번호를 다시 확인하세요.

"아직 준비 중입니다"라고 떠요

맨 처음 계산 도구를 받는 중입니다. 로그창에 "준비 완료"가 보일 때까지 5~10초만 기다렸다 다시 누르세요.

에러가 났어요 (빨간 ⚠ 메시지)

대개 입력 양식이 잘못 채워진 경우입니다. 성별이 남/여인지, 시간이 30분 단위인지, IN이 OUT보다 빠른지 확인하세요. 가장 깔끔한 방법은 빈 양식을 새로 받아 다시 채우는 것입니다.

결과가 마음에 안 들어요

시드 숫자를 다른 값으로 바꾸고 대진표 생성을 다시 누르면 전혀 다른 조합이 나옵니다. 몇 번 돌려 가장 좋은 걸 고르세요. 반복 횟수를 250~500으로 올리면 품질이 조금 더 좋아집니다(시간은 늘어남).

너무 느려요 / 멈춘 것 같아요

첫 로딩(10MB)만 느립니다. 휴대폰은 PC보다 느립니다. 반복 횟수가 크면(예: 1000) 오래 걸리니 150~250 정도로 낮춰 보세요.

내 명단이 외부로 새나가지 않나요?

안 나갑니다. 모든 계산은 여러분의 브라우저 안에서만 이뤄지고, 파일은 어떤 서버로도 전송되지 않습니다.

정상적으로는 빈 양식 받기 → 채우기 → 올리기 → 생성 → 다운로드, 이 흐름이면 끝입니다. 막히면 이 페이지의 해당 단계를 다시 보세요.
