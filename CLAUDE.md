# auto-tennis-bracket

테니스 클럽 주간 대진표 자동 생성 프로젝트.

## 하네스: 테니스 대진표

**목표:** 참가자 명단(이름·성별·구력·정/게·IN/OUT) + 코트 운영시간 → 30분 단위 슬롯×코트 매트릭스의 대진표 엑셀 출력.

**트리거:** "대진표", "대진 짜줘", "테니스 스케줄", "코트 배정", "참가자 명단으로 대진", "입력 양식", "테니스 입력 템플릿" 등 테니스 대진 관련 요청 시 `tennis-bracket-orchestrator` 스킬을 사용하라. 단순 질문(예: "openpyxl 어떻게 쓰지?")은 직접 응답 가능.

**워크플로우:**
- **Phase A (입력 템플릿 생성):** "양식 만들어줘" → `excel-renderer` 단독 호출
- **Phase B (대진표 생성):** 채워진 입력 엑셀 제공 → 4-에이전트 팀(`requirements-parser → bracket-composer ⇄ bracket-reviewer → excel-renderer`)

**중간 산출물:** `_workspace/01_parsed.json`, `02_bracket.json`, `03_review.json` — 재실행/부분 수정 시 활용. 새 실행 시 기존 `_workspace/`는 `_workspace_prev/`로 이동.

**규칙 요약 (전체 가이드는 오케스트레이터 스킬 참조):**
- 1게임 = 30분
- 남복/여복 우선, 인원·중복 시 혼복 (혼복은 같은 팀 남자 구력 ≥ 여자 구력)
- **코트 affinity**: A=여복/혼복 우선, B=남복 우선, C=무관
- **여성 07:30 이전 슬롯 회피** (IN 시간 우선)
- 같은 팀 페어 중복 회피, 3슬롯 연속 출전 최소화
- 동일 시간대 그룹 내 게임수 격차 ±1~2 목표
- 정회원/게스트 혼합 권장 (가중치 작음)

**통합 실행 스크립트:**
- `대진표_생성.py` (CLI) — `python 대진표_생성.py --in <입력엑셀> [--out <출력>] [--date <날짜>] [--seed N]`
- `대진표_생성.bat` (윈도우 더블클릭/CLI) — 대화형/인자형 모두 지원
- `--create-template [--prefill image]`로 빈/사전채움 입력 양식 생성

**파일 / 디렉토리 규칙:**
```
auto-tennis-bracket/
├── 대진표_생성.py / .bat   ← 실행 진입점
├── CLAUDE.md
├── 입력/                    ← 회원이 채울 양식
│   └── 테니스_입력양식.xlsx
├── 출력/                    ← 생성된 대진표 (날짜별 자동 명명)
│   └── 테니스_대진표_<YYMMDD>.xlsx
├── 샘플/                    ← 참고 자료
│   ├── 샘플_입력_13명.xlsx
│   └── 화면 캡처 *.png
├── _workspace/              ← 알고리즘 중간 산출물 (01_parsed/02_bracket/03_review.json)
└── .claude/                 ← 하네스 (agents/, skills/)
```

- `--in` 생략 시 자동으로 `입력/테니스_입력양식.xlsx` 사용
- `--out` 생략 시 자동으로 `출력/테니스_대진표_<날짜>.xlsx` (date 인자 또는 오늘)
- 사용자가 직접 만든 다른 양식이 있으면 `--in <경로>`로 명시 지정
