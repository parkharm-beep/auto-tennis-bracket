# Tennis Excel Output

`_workspace/01_parsed.json` + `_workspace/02_bracket.json`을 읽어 첨부 이미지 양식의 결과 엑셀을 생성.

## 출력 양식 (첨부 이미지 기반)

```
+---------+--------+--------+--------+--------+--------+--------+--------+----+------+----+
|         | 1번코트          | 2번코트          | 3번코트          |    참가자  게임수  |
+---------+--------+--------+--------+--------+--------+--------+--------+----+------+----+
| 1번게임 | 성함   | t1a t1b VS t2a t2b | ...     | ...     | 1 | 이름 | N |
| 07:00   |--------+--------+--------+--------+--------+--------+--------+    | (IN~OUT)|
| ~07:30  | 결과   |        |        |        |        |        |        |    |       |
+---------+--------+--------+--------+--------+--------+--------+--------+----+-------+
```

- 코트별로 다른 채움색 (예: 코트1=노랑 #FFFF99, 코트2=주황 #FFCC99, 코트3=하늘 #99CCFF)
- 매치 유형 표시: 혼복은 셀에 "혼" 표시 추가 (옵션)
- 게스트는 이름 뒤 "(G)" 표기
- 비어있는 슬롯(공석)은 회색 채움
- 우측 참가자 패널: 좌/우 두 그룹, 한 그룹당 10명 (총 20명까지). 그 이상이면 자동으로 확장.
- 참가자 셀 아래에 "IN시간~OUT시간" 작은 글씨 (예: "8시~12시")

## 사용

```powershell
python C:\Works\auto-tennis-bracket\.claude\skills\tennis-excel-output\scripts\render_bracket.py `
  --parsed  C:\Works\auto-tennis-bracket\_workspace\01_parsed.json `
  --bracket C:\Works\auto-tennis-bracket\_workspace\02_bracket.json `
  --out     C:\Works\auto-tennis-bracket\테니스_대진표_20260530.xlsx `
  --date    "26.5.30" `
  --title   "우리 테니스 클럽 이번 주 대진표"
```

## 셀 서식 규칙
- 폰트: 맑은 고딕 11pt, 이름은 굵게 10pt
- 시간/번호 셀: 가운데 정렬
- 이름 셀: 가운데 정렬, 코트색 배경
- "VS" 셀: 회색 가는 배경
- 결과 행은 흰색(비어있음)
- 외곽선: 모든 셀에 얇은 검정 테두리
- 컬럼 너비: 시간 7, 성함/결과 8, 이름 칸 10
- 행 높이: 성함/결과 각 22
